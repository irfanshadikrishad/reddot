# Crisis Disclaimer

RedDot is a device-first support tool. It can help prepare calls, messages, plans, reminders, and local records, but it cannot guarantee emergency response, message delivery, rescue, anonymity, confidentiality, or professional care.

If someone is in immediate danger, use the local emergency number or another trusted emergency service directly. If a screen, permission, or network step fails, use the fallback path shown in the app instead of assuming help has already been contacted.
